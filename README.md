# Pixlutter

The jolly file viewer for pictures by Clutter.

## Requirements

* [Clutter](https://wiki.gnome.org/Projects/Clutter)

### Install (Debian, Ubuntu)

    % sudo apt-get install libclutter-1.0-dev

## Usage

### Configure

    % ./autogen.sh
    % ./configure

### Build

    % make

### Run

    % ./pixclutter FILE

### Run test

    % test/run-test.sh
